# Payment Processing Microservices

This project demonstrates a simple microservices architecture using Java and Spring Boot.

## Services
1. Order Service
2. Payment Service

## Features
- REST API development using Spring Boot
- Microservices architecture
- Payment transaction simulation
- MySQL/PostgreSQL database integration
- API testing using Postman

## Tech Stack
- Java
- Spring Boot
- REST APIs
- Microservices
- MySQL / PostgreSQL
